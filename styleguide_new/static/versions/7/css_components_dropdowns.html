Dropdowns

npm install pui-css-dropdowns --save

Dropdowns are menus that can be triggered by buttons.

This is the basic dropdown.

  • Google
  • Yahoo
  • Aol 
<div class="dropdown">
  <button id="dropdown-button-1" class="dropdown-toggle btn-default-alt" type="button" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
    Button Dropdown
    <span class="caret"></span>
  </button>
  <div class="dropdown-menu">
    <ul role="menu" aria-labelledby="dropdown-button-1">
      <li role="presentation">
        <a href="http://www.google.com" role="menuitem">Google</a>
      </li>
      <li class="divider mvn"></li>
      <li role="presentation">
        <a href="http://www.yahoo.com" role="menuitem">Yahoo</a>
      </li>
      <li class="divider mvn"></li>
      <li role="presentation">
        <a href="http://www.aol.com" role="menuitem">Aol</a>
      </li>
    </ul>
  </div>
</div>

Dropdowns with dividers

If you want to add a divider between items in the dropdown, add a `li.divider` between list items.

  • Action
  • Another action
  • Something else here
  • Separated link 
<div class="dropdown dropdown-flat dropdown-link">
  <button id="drop3" data-toggle="dropdown" class="dropdown-toggle btn-link" type="button">
    <span>Link Dropdown</span>
    <span class="caret"></span>
  </button>
  <div class="dropdown-menu dropdown-menu-float">
    <ul role="menu" aria-labelledby="drop3">
      <li role="presentation">
        <a role="menuitem" tabindex="-1" href="https://twitter.com/fat">Action</a>
      </li>
      <li role="presentation">
        <a role="menuitem" tabindex="-1" href="https://twitter.com/fat">Another action</a>
      </li>
      <li role="presentation">
        <a role="menuitem" tabindex="-1" href="https://twitter.com/fat">Something else here</a>
      </li>
      <li role="presentation">
        <a role="menuitem" tabindex="-1" href="https://twitter.com/fat">Separated link</a>
      </li>
    </ul>
  </div>
</div>

Alignment

By default, a dropdown menu is automatically positioned 100% from the top and along the left side of its parent. Add .dropdown-menu-right to a .dropdown-menu to right align the dropdown menu

  • Google
  • Yahoo
  • Aol 
<div class="dropdown dropdown-flat">
  <button id="dropdown-button-1" class="dropdown-toggle btn-default-alt" type="button" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
    Button Dropdown
    <span class="caret"></span>
  </button>
  <div class="dropdown-menu dropdown-menu-float dropdown-menu-right">
    <ul role="menu" aria-labelledby="dropdown-button-1">
      <li role="presentation">
        <a href="http://www.google.com" role="menuitem">Google</a>
      </li>
      <li class="divider mvn"></li>
      <li role="presentation">
        <a href="http://www.yahoo.com" role="menuitem">Yahoo</a>
      </li>
      <li class="divider mvn"></li>
      <li role="presentation">
        <a href="http://www.aol.com" role="menuitem">Aol</a>
      </li>
    </ul>
  </div>
</div>

No links as dropdowns

Instead of using an a tag as a dropdown, use a button.btn.btn-link.

Here's a crazy-complex dropdown. Not for the faint of heart.

  • updates and alerts

    receive all tacos and burritos.

  • alerts only

    only receive tacos when burritos are issued for this product and files.

  • none

    do not receive any mexican food.
